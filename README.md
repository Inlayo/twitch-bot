# Twitch Live Notifier Bot

A simple bot that notifies your Discord server when your Twitch stream goes live:

- Monitors your Twitch channel for **live status**.
- Sends a **notification message** to a designated Discord channel automatically.
- Supports customizable messages and Discord webhook integration.

## Features

- Real-time Twitch live detection
- Automatic Discord notifications
- Customizable message templates
- Lightweight and easy to deploy (PM2 compatible)
